# Packly AI Solutions Landing Page

Static one-page website built outside the current repo at `C:\packly-ai-solutions-site`.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Local preview

Run this inside `C:\packly-ai-solutions-site`:

```powershell
python -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

This is a plain static site. There is no build step.

You can deploy the folder directly to:

- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages
- Any standard web host that serves static files

## Notes

- The visual direction follows the supplied pitch deck: black canvas, electric blue and violet ribbon light, large white headings.
- The layout was intentionally kept sharp and angular rather than rounded or pill-driven.
- Contact details currently match the pitch deck and can be edited directly in `index.html`.
